# Todo-Backend

[Todo-Backend](https://www.todobackend.com/) defines a simple REST API to showcase backend tech stacks.

This sample implements the specs using [Firestorm](https://github.com/connellw/Firestorm). You can run the sample locally and run the specs against it by opening [the test page](https://www.todobackend.com/specs/index.html?http://localhost:5000/todos/) targetting http://localhost:5000/todos/.
